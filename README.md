# MAERSK LINE

## Configuration

* browse to maersk folder in terminal
* check docker status
```bash
sh status.sh
```
* build docker images
```bash
sh build.sh
```
* init docker containers
```bash
sh init.sh
```
* cleanup images and containers
```bash
sh cleanup.sh
```

## Browse

* open browser and go to [localhost:8080](http://localhost:8080)
